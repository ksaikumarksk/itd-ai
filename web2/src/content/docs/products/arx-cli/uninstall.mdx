## Uninstallation

To uninstall arx-cli, you can use the following commands based on your package manager:

<Tabs>
  <TabItem label="npm">
    ```bash
  npm uninstall arx-cli
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
   yarn remove arx-cli
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm uninstall arx-cli
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash
    bun remove arx-cli
    ```
  </TabItem>
</Tabs>
